## 🌟 Projektbeschreibung

Diese Website bietet umfassende Informationen zu den Themen Organspende und Körperspende in einem interaktiven, benutzerfreundlichen Format. Die Plattform ist darauf ausgerichtet, komplexe medizinische und ethische Themen verständlich darzustellen und den Nutzern eine informierte Entscheidungsfindung zu ermöglichen.

Die Benutzer können zwischen den beiden Themenbereichen wechseln, die jeweils durch unterschiedliche Farbschemata gekennzeichnet sind:

- **Organspende** (blaues Farbschema) - Informationen zur Spende von Organen nach dem Tod oder zu Lebzeiten
- **Körperspende** (rotes/oranges Farbschema) - Informationen zur Spende des gesamten Körpers für die medizinische Ausbildung und Forschung

Die Website ist vollständig responsiv und bietet sowohl einen Light- als auch einen Dark-Mode für optimale Lesbarkeit unter verschiedenen Bedingungen.

## ✨ Hauptmerkmale

- **Interaktives Scroll-Storytelling**: Smooth Scrolling zwischen den Abschnitten mit animierten Übergängen durch Framer Motion
- **Themenwechsel**: Toggle zum einfachen Wechseln zwischen Organspende und Körperspende
- **Dark Mode**: Vollständige Unterstützung für helle und dunkle Benutzeroberfläche
- **Responsive Design**: Optimiert für alle Geräte von Mobiltelefonen bis Desktop-Computern
- **Timeline-Komponenten**: Visuelle Darstellung der Abläufe bei Organspende und Körperspende
- **Accordion-FAQ**: Übersichtliche Darstellung häufig gestellter Fragen zu beiden Themenbereichen
- **QR-Code-Generator**: Für den schnellen Zugriff auf den Organspendeausweis
- **Barrierefreiheit**: Entwickelt mit Fokus auf Zugänglichkeit für alle Nutzer

## 📚 Inhaltliche Struktur

### 🫀 Organspende

1. **Was ist Organspende?**
   - Grundlegende Informationen und Definitionen
   - Bedeutung und gesellschaftliche Relevanz

2. **Was kann gespendet werden?**
   - Übersicht über spendbare Organe (Herz, Lunge, Niere, Leber, etc.)
   - Informationen zu Gewebespenden

3. **Ablauf der Spende**
   - Diagnose Hirntod
   - Zustimmung / Ausweis
   - Organentnahme
   - Transport / Matching
   - Transplantation

4. **Gesetzliche Grundlagen**
   - Entscheidungslösung in Deutschland
   - Vergleich mit anderen Ländern
   - Rechtliche Rahmenbedingungen

5. **Voraussetzungen**
   - Alter und gesundheitliche Faktoren
   - Zustimmungsregelungen
   - Ausschlusskriterien

6. **Ethische Fragen**
   - Freiwilligkeit
   - Würde des Verstorbenen
   - Organknappheit
   - Verteilungsgerechtigkeit

7. **Pro & Contra**
   - Argumente für die Organspende
   - Bedenken und Gegenargumente

### 🧠 Körperspende

1. **Was ist Körperspende?**
   - Definition und Unterschied zur Organspende
   - Historischer Kontext

2. **Was passiert mit dem Körper?**
   - Anatomische Präparation
   - Nutzung in der Lehre
   - Forschungszwecke
   - Würdige Bestattung

3. **Ablauf der Körperspende**
   - Vertrag zu Lebzeiten
   - Tod und Überführung
   - Nutzung in der Anatomie
   - Bestattung und Gedenken

4. **Voraussetzungen & Bedingungen**
   - Schriftliche Vereinbarung
   - Ausschlusskriterien
   - Keine gleichzeitige Organspende
   - Räumliche Faktoren

5. **Ethische Fragen**
   - Respekt und Würde
   - Umgang mit dem Tod
   - Angehörige und Trauerarbeit
   - Freiwilligkeit und Aufklärung

6. **Wissenschaftlicher Nutzen**
   - Ausbildung von Medizinern
   - Entwicklung neuer OP-Techniken
   - Erforschung von Krankheiten
   - Entwicklung medizinischer Geräte

### 🤔 Gemeinsame Abschnitte

- **FAQ**: Häufig gestellte Fragen zu beiden Themenbereichen
- **Call to Action**: Informationen zur Entscheidungsfindung und Links zu offiziellen Stellen
- **QR-Code**: Für den schnellen Zugriff auf den Organspendeausweis

## 🔧 Technologien

Das Projekt wurde mit modernen Webtechnologien entwickelt:

- **[Next.js](https://nextjs.org)** - React Framework für serverseitiges Rendering und optimierte Performance
- **[React](https://reactjs.org/)** - JavaScript-Bibliothek für Benutzeroberflächen
- **[TypeScript](https://www.typescriptlang.org/)** - Typsicherheit und verbesserte Entwicklererfahrung
- **[TailwindCSS](https://tailwindcss.com/)** - Utility-First CSS-Framework für schnelles Styling
- **[ShadCN UI](https://ui.shadcn.com/)** - Wiederverwendbare UI-Komponenten
- **[Framer Motion](https://www.framer.com/motion/)** - Bibliothek für Animationen und Übergänge
- **[next-themes](https://github.com/pacocoursey/next-themes)** - Theme-Unterstützung für Next.js (Dark Mode)

## 🚀 Installation und Start

Folgen Sie diesen Schritten, um das Projekt lokal einzurichten und zu starten:

```bash
# Repository klonen
git clone https://github.com/ttvtimotheus/organ-body-donation.git

# In das Projektverzeichnis wechseln
cd organ-body-donation

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev
```

Öffnen Sie [http://localhost:3000](http://localhost:3000) in Ihrem Browser, um die Website anzusehen.

### Systemvoraussetzungen

- Node.js 18.17 oder höher
- npm 9.6.7 oder höher

## 🎨 Design und Benutzeroberfläche

Das Design der Website folgt modernen UI/UX-Prinzipien:

- **Farbschemata**:
  - Organspende: Blaue Farbpalette (beruhigend, vertrauenswürdig)
  - Körperspende: Rote/Orange Farbpalette (warm, würdevoll)

- **Typografie**:
  - Klare, gut lesbare Schriftarten
  - Hierarchische Textstruktur für bessere Lesbarkeit

- **Animationen**:
  - Sanfte Übergänge zwischen Abschnitten
  - Subtile Animationen für interaktive Elemente

- **Komponenten**:
  - Cards für strukturierte Informationsdarstellung
  - Timeline für prozessorientierte Abläufe
  - Accordion für FAQ-Sektion
  - Toggle für Themenwechsel
